# **App Name**: e-Rotary Hub

## Core Features:

- Campaign Carousel: Display live campaign banners in a carousel format, with Rotary-blue overlays.
- Blog Section: Implement a clean and modern blog section with an editor-style CMS interface for submitting posts. The posts can be sorted with tags, and have social media sharing buttons.
- Event Calendar: Display upcoming events with an RSVP button and live Zoom/YouTube embed support, including an 'Add to Calendar' feature.
- AI Content Summarizer: Integrate an AI tool to summarize long campaign descriptions into short highlights to improve user understanding. This LLM powered tool will use reasoning to decide the best summarization.
- Idea Box: Allow users to suggest project ideas and vote on which should be tackled next to gather community feedback and drive platform improvement.
- User Impact Dashboard: Provide each user with a personal dashboard to track impact, campaigns joined, and followed creators and rooms. The dashboard includes milestones and a volunteering record.

## Style Guidelines:

- Primary color: Rotary Royal Blue (#002B7F) to evoke a sense of trust and authority. This deep blue is the core brand color of Rotary International.
- Background color: Desaturated Rotary Royal Blue (#E5E8EF) in a light color scheme to give a cool and spacious atmosphere that keeps users focused on the content.
- Accent color: Rotary Gold (#F7A31B), strategically used for interactive elements and CTAs to draw attention and signal importance.
- Recommended font pairing: 'Space Grotesk' (sans-serif) for headers to reflect strength and modernity, paired with 'Inter' (sans-serif) for body text to maintain clarity and readability. 'Source Code Pro' will be used for code snippets, if needed.
- Use clean cards with rounded corners to display content and campaigns in a structured manner.
- Implement a sticky bottom navigation for mobile responsiveness with key actions: Home, Chat, Campaigns, Create, Profile.
- Incorporate subtle page transitions to give a futuristic feel and improve user engagement. These can be simple fades or slides.
